Fornisce classi che svolgono diverse funzioni di integrazione dei servizi applicativi con la Porta di Dominio OpenSPCoop. Le classi implementano le interfacce IGestoreIntegrazionePD o IGestoreIntegrazionePA per l'integrazione lato Porta Delegata o lato Porta Applicativa. Le implementazioni fornite per entrambe le modalita' sono:
  • Soap: Gestisce un header di integrazione SOAP
  • Trasporto: Gestisce proprieta' del trasporto
  • UrlBased: Gestisce proprieta' di una url in stile form-based
