Fornisce classi che svolgono funzioni di controllo di soglie su risorse del sistema (FileSystem, DB). Le classi implementano una interfaccia della classe IThreshold utilizzabile da OpenSPCoop per effettuare il controllo. Implementazioni esistenti sono fornite per il database MySQL con la classe MySQLThreshold
