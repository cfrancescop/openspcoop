Gestione della configurazione di OpenSPCoop (xml o DB). Nella configurazione sono registrate le seguenti informazioni :
  • Porte Delegate.
  • Porte Applicative
  • Dati della porta di dominio
  • Servizi Applicativi

All'interno di questo package sono definite classi che sono il mapping delle informazioni presenti nel file di configurazione.

Vengono fornite classi per la gestione delle risorse esterne utilizzata da OpenSPCoop:

  • DBManager: gestisce le connessioni al DB
  • QueueManager: gestisce le connessioni al broker JMS
Altre classi fornite gestiscono pool alle risorse DB e JMS.

Vengono forniti reader di file properties utilizzati da OpenSPCoop:

  • OpenSPCoopProperties: gestisce le proprieta' di configurazione di OpenSPCoop
  • ClassNameProperties: gestisce i nomi di classi istanziate dinamicamente da OpenSPCoop
